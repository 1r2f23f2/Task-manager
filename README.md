## Описание

Random Task Generator — это графическое приложение на Python для генерации случайных задач. Приложение помогает организовать вашу повседневную деятельность, предлагая случайные задачи из разных категорий.

### Основные возможности:

- ✅ Генерация случайных задач из предопределённого списка
- ✅ Добавление своих задач с указанием категории
- ✅ Валидация ввода (нельзя добавить пустую задачу)
- ✅ Фильтрация истории по категориям (учёба, спорт, работа)
- ✅ Сохранение истории сгенерированных задач в JSON
- ✅ Автоматическая загрузка истории при запуске
- ✅ Красивый и интуитивно понятный интерфейс на tkinter

## Технологии

- Python 3.7+
- tkinter (встроенная библиотека)
- json (встроенная библиотека)
- random (встроенная библиотека)
- datetime (встроенная библиотека)
